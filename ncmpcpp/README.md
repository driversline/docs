> Start:
 * `mpd &`
 * `ncmpcpp`

> Example:
 * [bspwm](https://github.com/driversline/bspwm/tree/main/dotfiles/player)

> Path:
 * `$HOME/.config/mpd`
 * `$HOME/.config/ncmpcpp`
 * `$HOME/Music (Your)`
   
> Configs:
 *`mpd.conf` and `database`
 * `config`
 * `*.mp3`

> PWD:
 * `$HOME/.config/mpd/mpd.conf | $HOME/.config/mpd/database`
 * `$HOME/.config/ncmpcpp/config`
 * `$HOME/Music/*.mp3`
