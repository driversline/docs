# Запуск

```sh
mpd &
```

```sh
ncmpcpp
```

# Пути к конфигам
 * `$HOME/.config/mpd`
 * `$HOME/.config/ncmpcpp`
 * `$HOME/Music (Твой)`
   
# Название конфигов
 * `mpd.conf` и `database`
 * `config`
 * `*.mp3`

# Pwd
 * `$HOME/.config/mpd/mpd.conf | $HOME/.config/mpd/database`
 * `$HOME/.config/ncmpcpp/config`
 * `$HOME/Music/*.mp3`
