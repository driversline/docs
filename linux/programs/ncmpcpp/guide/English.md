# Launch

```sh
mpd &
```

```sh
ncmpcpp
```

# Paths to configs
 * `$HOME/.config/mpd`
 * `$HOME/.config/ncmpcpp`
 * `$HOME/Music (Your)`
   
# Names of configs
 *`mpd.conf` и `database`
 * `config`
 * `*.mp3`

# Pwd
 * `$HOME/.config/mpd/mpd.conf | $HOME/.config/mpd/database`
 * `$HOME/.config/ncmpcpp/config`
 * `$HOME/Music/*.mp3`
